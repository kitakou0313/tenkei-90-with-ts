- N = 4...2組
    - {()}
    - {}()
- N = 6...3組
    - {(())}
    - {()()}
    - {}(())
    - {}()()
    - (()){}
    - ()(){}
- ((())(())) のようなケースが生成できなかった 2組以上のカッコのペアで生成される文字列
