## 考察
- 制約
    - D, N <= O(10^2)
    - Ai, Bi <= O(10^1)
    - Ci <= O(10^2)
- 要約
    - D日間の間、N枚の服を着る
        - 何度も選んでよく、選ばなくても良い
    - 服iは以下の条件の時にD日に着ることができる
        - D日の気温が服iの適切な気温Ai ~ Biの時
    - D日間の服の着方のうち、以下の値が最大になるものを求める
        - |0日目の服の派手さ - 1日目の服の派手さ| + |1日目の服の派手さ - 2日目の服の派手さ| ... D日目まで
- 考察
    - 愚直解
        - N^Dの組を求める -> 条件を満たすか確認 -> 最大のものを探索
        - 計算量
            - O(N^D * 10^2)
    - DPで考える
        - 部分構造の最適性
            - ある？
                - 前日までの最大の結果 + 本日の最大値
        - 部分問題の特徴づけ
            - 
    - 考えるべき区間はN-1個
