# 問題リンク
- https://atcoder.jp/contests/arc054/tasks/arc054_b

## 考察
- 前提
    - 制約
        - P <= 10^18
    - コンピューターの計算時間はx年後に以下のように変化する
        - 2^(x/1.5)
- 要約
    - x + P * (2^(-x/1.5))の最小化
- 方針
    - 二分法で探索し、得られた結果をPと比較して小さい方を出力
    - 区間の考察
        - 最大
            - P
                - Pより大きい年数は考えなくていいため
                    - P以降の年に最低値をとっても、0年目から計算を始めてP年に完了するケースに確実に負けるため
        - 最小
            - 0
                - 負の年数は取れないため
                    - 0年目が最小 -> Pとイコール
    - 導関数による場合分け
        - 0 ~ Pの間で
            - 0をとる
                - その点が最小値
            - 常に>0
                - 0が最小値
                    - 単調増加なので
            - 常に<0
                - Pが最小値
                    - 単調減少なので
