## 考察
- 制約
    - m, n <= 90
        - m*n = 810 O(10^2)
    - 氷のある区画が1, ない区画が0
- 要約
    - 上下左右の区画かつその区画の値が1の時、遷移できる
    - 移動可能な区画がなくなるまで移動した時、移動した最大の区画数を求める
- 考察
    - 1区画ずつ、そこを起点にしてDFS
        - 実装詳細
            - m, n の各区画について以下を繰り返す
            - 処理
                - 引数で指定された区画についての処理
                    - その区画を探索済みにする
                    - 通過した区画数カウントを増加
                        - 上の探索済み区画セットのサイズの増加と同義なのでスキップも可
                    - 現在の最大値カウントと比較して大きい方を採用
                - 隣接区画について以下を処理
                    - 0 ~ m-1, 0 ~ n-1の範囲ではない
                        - continue
                    - 探索済み または または0の場合
                        - continue
                    - この処理を再帰的に呼び出す
                - 区画の処理後、以下を行う
                    - 探索済みから削除
                    - カウント数を-1
        - 計算量
