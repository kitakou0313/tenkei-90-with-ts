- https://atcoder.jp/contests/abc023/tasks/abc023_d

## 考察
- 制約
    - N <= 10^5
    - Hi, Si <= 10^9
- 問題の要約
    - 風船を割った時に受けるペナルティを最小にする
    - 風船
        - 開始時の高度はHi
        - 1秒経過するごとにSiだけ高度が上がる
    - 風船の割り方
        - 競技開始時 + 1秒につき1つ割っていく
        - 全風船を割る
    - 受けるペナルティ
        - 風船を割った時の高度
    - 割った全風船によるペナルティのうち、最大値を得点とする
- 考察
    - 全通り
        - O(N!)でTLE
    - 2分探索を考える
        - 可能な得点と不可能な得点の境界を求める
            - Hi + Si * Nの最大値が可能な最大の得点
                - 間の得点は各風船のHi + Si*nでnを0 ~ N-1で変化させたもの
            - 0が不可能な最小の得点
            - 考察
                - 得点の最小値が見つかった時、それより大きいものは全て可能になる？
