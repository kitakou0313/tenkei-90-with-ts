## 考察
- 制約
    - N <= 10^3
    - M <= 2*10^8
    - Pi <= 10^8
        - 1 <= i <= N
- 問題の要約
    - Piから0 ~ N個まで要素を選びそれらの合計を考える。この時合計がM以下となる要素の選び方のうち最大値は何か
- 考察
    - 全探索
        - 全てのPiの選び方を全探索
            - Bit全探索でN個の選び方を全探索する
        - O(2^(N))でTLE
    - 0 ~ Mの範囲の値について、合計がその値になる選び方が存在するか探索する
        - Nの組み合わせを見ることになり厳しい
