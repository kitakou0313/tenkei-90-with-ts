## 考察
- 制約
    - K <= N <= 15
    - a_i <= 10^9 
        - i < N
- 問題
    - a_j < a_iを満たすa_iがK個以上存在するようにa_iに値を加算する時に必要な最小の追加分
        - j < i
- 考察
    - 2^15 = 32768（10^4オーダー）
    - N = Kの時（小課題1）
        - あるa_iについて、j < iとなるa_jを全て確認し、
            - a_i <= a_jとなるa_jがあれば、a_i = a_j + 1に更新
            - なければそのまま
        - とする
    - a_iを見える状態にしたい場合は、j < iとなるa_jを全て確認し以下の操作をすれば良い
        - a_i <= a_jとなるa_jがあれば、a_i = a_j + 1に更新
        - なければそのまま
        - a_j + 1より大きい高さにするケースは考えなくて良い
            - a_j + 1を選んだケースより確実に大きくなるため
