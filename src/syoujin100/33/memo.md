# 問題リンク
- https://atcoder.jp/contests/abc088/tasks/abc088_d

## 考察
- 制約
    - H,W <= 50
- 要約
    - あるグリッドから上下左右の白グリッドに移動できる
    - 迷路を解く前、グリッドのマスの白 -> 黒に変更することができる
    - (1, 1)から(H-,1 W-1)にゴールした時、白 -> 黒に変更したグリッドの数が得点となる
    - この得点を最大化する
        - ゴールできない時は-1
- 考察
    - 経路は最短経路だけ考えれば良い
        - それ以外の経路を通ったケースでは得点が超えることは無い
    - 得点の範囲
        - H * W - 2 - 黒マス ~ 0
            - スタートとゴールは色変できない
    - 愚直解
        - 最短経路を求める -> それ以外の白マスを黒に変更
    - 実装考察
        - BFSで(H-,1 W-1)までの経路を求める
            - Queueを初期化
            - 探索済み座標管理Setを初期化
            - 座標ごとの最短経路記録Mapを初期化
                - 座標をキー、(0, 0)からの最短距離の座標が配列
            - 座標ごとの最短経路記録Mapに(0, 0)を追加
                - 
            - Queueに(0, 0)を追加
            - (0, 0)を探索済み座標管理Setに追加
            - Queueが空になるまで以下を繰り返し
                - Queueから座標を出す
                - 4方向の隣接座標について以下の処理
                    - 座標がgiridsの範囲内か判定
                    - 白グリッドか判定
                    - 座標ごとの最短経路記録Mapに追加
                        - 現在の座標までの最短経路 + 隣接座標
                    - Queueに追加
                    - 探索済み座標管理Setに追加
        - 得点計算
            - (H-,1 W-1)に到達できていたら
                - 最短経路記録Mapに(H-,1 W-1)があるかで判定
                - 全白グリッドの数と座標を記録
                    - Setに入れる
                - 白グリッドの数 - 経路上の白グリどの数 を print
                    - 全白グリッドの座標のSetから経路上白グリッドを除く
            - できていなければ
                - -1 を print
