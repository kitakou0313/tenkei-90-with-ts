# 問題リンク
- https://atcoder.jp/contests/joi2011yo/tasks/joi2011yo_d

## 考察
- 制約
    - N <= 10^2
- 要約
    - 数列の空欄のうち、
        - 最も右側のものに = を
        - それ以外のものに+, -を
    - 入れる。
    - この空欄への演算子の入れ方のうち、以下を満たすものの数を求める
        - 等式として成立している
        - 計算途中で現れる数が 0以上 20以下の範囲になる
- 考察
    - 愚直解
        - 全ての空欄に可能な演算子（+, -）を入れる -> 検証 を行って個数カウント
        - 計算量
            - 0(2^(N-1))でTLE
## メモ
- 空白の個数カウント -> N - 1
    - 1 2 3 4 5
    - 演算子を入れられるのはN - 1 - 1（最も右側のものは=が入るため演算子が入れられない）
