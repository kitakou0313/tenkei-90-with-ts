# 問題リンク
- https://atcoder.jp/contests/joi2007ho/tasks/joi2007ho_c

## 考察
- 制約
    - N <= 3000 = 10^3
        - Nは柱の本数
- 出力
    - 与えられた柱で構成される正方形のうち、面積が最大のものを返す
    - 正方形が存在しなければ0を返す
- 全探索
    - 柱N本から4つ選び、面積が最大のものを探す
        - N_C_4 -> O(10^12)でTLE
- 探索する候補を減らす
    - 二つの柱を選び、それを元に正方形が作れるか考える
        - 2柱選べば推移的に正方形が考えられる
