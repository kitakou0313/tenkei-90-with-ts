## 考察
- 制約
    - N <= 10^2, K <= N
- 要約
    - 条件を満たすパスタの選び方の数を求める
    - 条件
        - 与えられたK日分は指定されたパスタソースを選ぶ
        - 3日以上同じパスタが連続しない
- 考察
    - 愚直解
        - 各日にちごとに3通り入れ、得られた組み合わせについて条件を満たすか考える
        - 時間計算量
            - O(3^N) でTLE
    - DPでとく
        - 3日連続しない = 8パターン
