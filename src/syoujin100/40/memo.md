# 問題リンク
- https://atcoder.jp/contests/joi2012yo/tasks/joi2012yo_d

## 考察
- 制約
    - N <= 10^2, K <= N
- 要約
    - 条件を満たすパスタの選び方の数を求める
    - 条件
        - 与えられたK日分は指定されたパスタソースを選ぶ
        - 3日以上同じパスタが連続しない
- 考察
    - 愚直解
        - 各日にちごとに3通り入れ、得られた組み合わせについて条件を満たすか考える
        - 時間計算量
            - O(3^N) でTLE
    - DPでとく
        - 3日連続しない = 27 - 3パターン

## メモ
- DPの配列のキーは部分問題の特徴と同じ
- 問題の制約を緩めて解けるか考える
    - how to solve problemに書いてあったのと同じ
- dpは必要な情報を取得できるためのキーを如何に設計できるか ということっぽい
